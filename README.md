# Meeting Burner 🚀

![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white)
![AdSense Ready](https://img.shields.io/badge/AdSense-Ready-success?style=for-the-badge&logo=google)

> A sleek, distraction-free web tool designed to help professionals track meeting costs in real-time—calculating everything from dollars spent to their equivalent value in used cars.

## ✨ Features

* **Real-Time Cost Tracking:** Calculate the "burn rate" of any meeting based on attendee count and average salary.
* **Humorous Insights:** See the meeting cost converted into fun metrics (like Used Honda Civics).
* **Lightning Fast:** Built with pure, semantic HTML for minimal load times.
* **SEO Optimized:** Pre-configured with `sitemap.xml` and `robots.txt` for search engines.
* **AdSense Ready:** Structured to accommodate Google AdSense units for easy monetization.

## 🛠️ Tech Stack

* **HTML5/CSS3:** Core structure and responsive UI.
* **JavaScript:** Real-time calculation logic and timer functionality.

## 🚀 Getting Started

### Prerequisites
You just need a modern web browser to view the site.

### Installation

1. **Clone the repository:**
   `git clone https://github.com/Emmy81/meeting-burner.git`

2. **Navigate to the project directory:**
   `cd meeting-burner`

3. **Launch:**
   Open `index.html` directly in your browser.

## 💰 Monetization (AdSense Setup)

This project is built to be AdSense ready. To activate ads:

1. Ensure your site is approved in your **Google AdSense** dashboard.
2. Place your auto-ads script snippet inside the `<head>` tag of `index.html`.
3. For manual units, place the ad `<div>` codes into the designated UI containers.
